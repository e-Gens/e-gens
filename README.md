# e-Gens - Sistema de Gestão do Ensino

## Project setup

```shell
yarn install
```

### Compiles and hot-reloads for development

```shell
yarn run serve
```

### Compiles and minifies for production

```shell
yarn run build
```

### Lints and fixes files

```shell
yarn run lint
```
